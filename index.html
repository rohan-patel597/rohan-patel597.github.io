Projects

Contoso Sales Analysis

This Power BI dashboard offers a deep dive into Contoso Inc.'s sales data, encompassing over 2.28 million rows of transactions. Through rigorous data cleaning and insightful analysis, the dashboard highlights key sales trends, channel performance, product mix, and discount strategies. By addressing critical business questions and leveraging data-driven insights, the analysis supports strategic decision-making across sales, marketing, product, and operations teams.

Power BI Spotify Top Music Artist Analysis

This Power BI project offers an interactive and dynamic dashboard that analyzes top Spotify songs from 2018 to 2023. Leveraging advanced visualization tools like DENEB and incorporating custom HTML content, the dashboard provides in-depth insights into music trends, artist popularity, and song characteristics. This project showcases my expertise in data visualization and analytics, delivering valuable insights into the evolving landscape of popular music.

Smart Cartridge Masking: Computer Vision

This computer vision project revolutionizes forensic ballistics by automating the analysis of fired bullet cartridges. Using advanced image processing techniques, including edge detection, Hough Circle Transform, and dynamic masking, the custom algorithm accurately identifies key forensic features like breach face impressions and firing pin marks. This project enhances the speed and accuracy of criminal investigations, showcasing my expertise in computer vision, image processing, and algorithm development.
